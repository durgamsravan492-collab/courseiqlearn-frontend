# Course Purchase Flow UI

Minimal production-ready scaffolding for the Course Purchase Flow UI.

Quick commands

Install dependencies:
```bash
npm install
```

Run development server:
```bash
npm run dev
```

Typecheck:
```bash
npm run typecheck
```

Build for production:
```bash
npm run build
```

Preview production build (local):
```bash
npm run serve
```

Lint and format:
```bash
npm run lint
npm run format
```

Environment

Copy `.env.example` to `.env` and update `VITE_BASE_URL` or other VITE_ variables as needed.

Notes

- This repo was reorganized for production readiness: TypeScript strict mode, Vite build optimizations, ESLint + Prettier, and CI workflow.
- Replace `public/favicon.ico` with your real binary favicon.
